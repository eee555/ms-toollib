# 开发计划

计划开发：

- 兼容更多的扫雷录像格式。
- 更多的优秀的指标、录像分析算法。
- zini的算法、更好的最少点击次数的算法。
- 在java等其他语言端的绑定。
- 使用强化学习的扫雷ai。
- 其他算法的优化。

而不会开发：

- 与ui、视觉相关的功能。
- 扫码收费以解锁某功能的功能。

# 国际化

- 项目接受任何语言的注释和文档，只要符合开发人员的习惯。



# Development Plan

Planned developments:

* Support for more replay formats.
* Additional high-quality metrics and replay-analysis algorithms.
* Implementation of Zini’s algorithm and improved minimum-click algorithms.
* Bindings for Java or other languages.
* A reinforcement-learning–based Minesweeper AI.
* Optimization of other algorithms.

Will not be developed:

* UI or visually related features.
* Any “scan to pay to unlock features” functionality.

# Internationalization

* The project accepts comments and documents in any language, as long as they suit the habits of the developers.
